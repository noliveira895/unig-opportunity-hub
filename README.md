# UNIG Opportunity Hub

Sistema web fullstack desenvolvido como atividade do Tema Integrador do 3º período do curso de Análise e Desenvolvimento de Sistemas da UNIG.

---

## 📌 Descrição do Projeto

O UNIG Opportunity Hub é uma plataforma web para cadastro e divulgação de oportunidades acadêmicas e profissionais na área de tecnologia.

O sistema permite o gerenciamento completo de oportunidades através de um CRUD:

- Criar oportunidades
- Listar oportunidades
- Editar oportunidades
- Excluir oportunidades

---

## 🎯 Objetivo

Facilitar o acesso de estudantes e jovens da área de tecnologia a oportunidades como cursos, estágios, empregos e mentorias, centralizando informações em um único sistema simples e acessível.

---

## 🌍 ODS Relacionado

ODS 9 — Indústria, Inovação e Infraestrutura

O projeto contribui para:

- Inclusão digital
- Acesso à informação
- Uso de tecnologia para solução de problemas sociais
- Incentivo à inovação
